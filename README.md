# ILF — Impact Life Foundation

**A modern, bilingual (English + বাংলা) portfolio website for ILF (Impact Life Foundation), a non-political social organization from Bangladesh.**

Built with **Next.js 16 (App Router)**, **React 19**, **Tailwind CSS v4**, and **TypeScript**.

---

## Features

- **Bilingual** — English and Bangla (বাংলা), with a language switcher (EN | বাংলা)
- **Native i18n routing** — `/en/*` and `/bn/*` URL structure via Next.js App Router
- **7 Pages** — Home, About, Activities (10 programs), Gallery, Team, Get Involved, Contact
- **API Routes** — Contact and Volunteer form endpoints with Zod validation
- **SEO-optimized** — metadata, Open Graph, html lang attribute
- **Fully responsive** — mobile, tablet, desktop
- **Accessible** — semantic HTML, ARIA labels, keyboard-navigable
- **Green + White theme** — representing trust, growth, and peace
